◆ get()

ASTBase * ASTContext::get ( const str &  name)

look up the variable table in the current and parent scopes

This function starts by searching the current scope. If the target is not found in current scope, search the parent scope, repeat the process until found. Return nullptr if not found in all visible scopes.
